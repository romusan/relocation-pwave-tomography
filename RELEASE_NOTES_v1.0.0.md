# Version 1.0.0 - Materials for manuscript review

This release contains the reproducibility package associated with the manuscript:

**Fast-marching earthquake relocation and P-wave tomography constrain a seismicity-defined Nazca Benioff zone beneath the Middle Magdalena Valley, Colombia**

Included materials:

- Manuscript source, compiled PDF, and BibTeX references.
- Final Python scripts used for relocation, tomography diagnostics, full current-dataset 1-D starting-model sensitivity, uncertainty analysis, RMS-improvement examples, and Benioff-surface reconstruction.
- Processed CSV/JSON tables supporting the manuscript results.
- Final figures used in the manuscript and additional diagnostic figures.
- Metrics files summarizing resolution tests, regularization diagnostics, 1-D starting-model sensitivity, station statics, Slab2 comparison, and uncertainty estimates.

Large raw SGC XML event files and waveform downloads are not included. They can be obtained from the public Servicio Geologico Colombiano services using the scripts and selection tables provided in this repository.
